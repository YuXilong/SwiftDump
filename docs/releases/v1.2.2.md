# SwiftDump v1.2.2

本版本完成 CLI 作者署名修正和 GitHub/Homebrew 自动发布，并修复回归 fixture 在更新版 Swift 6 严格并发检查下的构建问题。

## 主要变化

- `SwiftDump --version` 输出 `Created by yuxilong`。
- 推送 `vX.Y.Z` tag 后自动构建、静态分析、回归、签名、打包并发布 GitHub Release 和完整日志。
- Homebrew tap 每小时自动发现最新正式 Release，严格核对 ZIP、`SHA256SUMS` 和 GitHub asset digest。
- Formula 自动更新前执行样式检查、在线审计、真实安装、Formula 测试、签名及 Universal 架构验证。
- 删除 regression fixture 中不必要的跨 actor `Task` 调用，兼容更新版 Swift 6 `sending` data-race 检查；被 dump 的 async 方法声明和符号保持不变。

## Homebrew

```sh
brew update
brew upgrade yuxilong/tap/swift-dump
```

安装后同时提供 `SwiftDump` 和 `swift-dump` 两个命令。
