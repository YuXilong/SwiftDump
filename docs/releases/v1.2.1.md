# SwiftDump v1.2.1

本版本修正 CLI 作者署名，并将源码 Release 与 Homebrew Formula 发布流程自动化。

## 主要变化

- `SwiftDump --version` 的作者信息更新为 `Created by yuxilong`。
- 推送 `vX.Y.Z` tag 后，GitHub Actions 自动完成 Release 构建、静态分析、Swift 5/6 回归、Universal 打包、ad hoc 全架构签名、日志和 SHA-256 发布。
- `YuXilong/homebrew-tap` 每小时自动检查最新正式 Release，验证 ZIP 与 `SHA256SUMS` 一致后更新 Formula。
- Homebrew 自动更新在提交前执行 `brew style`、`brew audit`、安装和 Formula 测试。

## Homebrew

```sh
brew update
brew upgrade yuxilong/tap/swift-dump
```

安装后同时提供 `SwiftDump` 和 `swift-dump` 两个命令。

## 发布附件

- `SwiftDump-v1.2.1-macos-universal.zip`
- `SwiftDump-v1.2.1-build.log`
- `SwiftDump-v1.2.1-regression.log`
- `SHA256SUMS`

发布二进制使用 ad hoc signature，未进行 Apple notarization。
